Minhas habilidades

Essas são algumas habilidades que domino e pratico que permitem criar meus projetos.

  • HTML

    HTML (Hypertext Markup Language) é a linguagem padrão para criar e estruturar páginas web. Ele usa tags (etiquetas) para definir elementos como títulos, parágrafos, links, imagens, listas e tabelas.

  • CSS

    CSS (Cascading Style Sheets) é uma linguagem usada para estilizar e formatar páginas web. Ele controla o layout, cores, fontes, espaçamentos e outros aspectos visuais de um site. O CSS funciona em conjunto com o HTML, aplicando estilos aos elementos HTML.

  • JavaScript

    JavaScript (JS) é uma linguagem de programação usada para adicionar interatividade e comportamentos dinâmicos a páginas web. Ela permite manipular o conteúdo HTML, estilos CSS e responder a eventos do usuário, como cliques, movimentos do mouse e entradas de teclado. JavaScript é uma das três principais tecnologias da web, ao lado de HTML e CSS.

  • React

    React é uma biblioteca JavaScript criada pelo Facebook para construir interfaces de usuário (UI) de forma eficiente, especialmente para aplicações de página única (SPA - Single Page Applications). Ele permite criar componentes reutilizáveis e gerenciar o estado da aplicação de maneira declarativa, facilitando o desenvolvimento de interfaces dinâmicas e interativas.

  • Git

    Git é um sistema de controle de versão distribuído amplamente utilizado para rastrear mudanças em arquivos e coordenar o trabalho entre várias pessoas em projetos de software. Ele permite que você salve versões do seu código, reverta para versões anteriores, crie branches (ramificações) para desenvolver novas funcionalidades e colabore com outras pessoas de forma eficiente.
